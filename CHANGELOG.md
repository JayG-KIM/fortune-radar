# Changelog

## [2.0.0] - 2025-02-23
### 🚀 Major Changes
- LLM 기반 → 템플릿 기반 엔진으로 전환 (API 비용 0원!)
- 업무지구 날씨 선택 기능 (CBD/YBD/GBD/BBD/MBD)

### ✨ New Features
- 띠×별자리 궁합 (144개 조합)
- 시간대별 운세 변화 (출근길/오전/점심/오후/퇴근후)
- 오늘의 변수 (40개 랜덤 풀)
- 연휴 전날 자동 감지
- 양력+음력 생일 체크
- 직장인 특수일 (월초/월말/분기말/연초/연말)
- 계절별 감성 레이어
- 날씨별 점심 팁
- 행운템 (30종 + 띠별 연결 이유)

### 🗑️ Removed
- Gemini API 의존성 제거
- 429 에러 화면 (더 이상 필요 없음)
- streamlit-lottie 의존성 제거
